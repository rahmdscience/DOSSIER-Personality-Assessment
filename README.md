# DOSSIER: Personality Assessment

## Overview

**Personality Assessment** adalah aplikasi web interaktif berbasis kuis psikometri yang digunakan untuk mengidentifikasi tipe kepribadian pengguna berdasarkan analisis multidimensi.

Aplikasi ini menggabungkan pendekatan logika kuis dengan tampilan visual bertema **Neo-Brutalism Industrial + Blueprint System**, menciptakan pengalaman seperti sistem analisis dalam departemen intelijen.

---

## Core Concept

Aplikasi ini menggunakan 4 dimensi utama:

1. **Fokus Energi**
   - Ekstrovert (E)
   - Introvert (I)

2. **Perolehan Informasi**
   - Realis (R)
   - Intuitif (N)

3. **Pengambilan Keputusan**
   - Logis (L)
   - Perasa (F)

4. **Gaya Hidup**
   - Terstruktur (T)
   - Spontan (S)

Total kombinasi menghasilkan berbagai tipe seperti:
- IRLT
- ENFS
- IRNF
- ERNT
- dll

## Features

### Kuis Interaktif
- 30 pertanyaan sistematis
- pilihan jawaban A/B
- progres bar real-time
- indikator persentase

### Hasil Analisis
- kode kepribadian
- nama tipe
- deskripsi lengkap
- rekomendasi karir
- gaya komunikasi
- sinergi tim

### Visualisasi Data
- radar chart menggunakan Chart.js
- interaktif (klik label → detail trait)
- responsif (mobile & desktop)

### UI/UX
- Neo-Brutalism style
- blueprint grid background
- hard shadow berlapis
- border tebal tanpa radius
- tipografi Courier Prime & Crimson Text

### Sistem Interaktif
- accordion arsip hasil
- modal penjelasan trait
- share ke:
  - Twitter
  - Facebook
- copy hasil ke clipboard
- notification system

## How To Run

1. Download / clone repository
2. Pastikan file:
   - index.html
   - style.css
   - script.js
   berada dalam satu folder
3. Buka `index.html` di browser

Disarankan menggunakan Live Server di VS Code.

---

## How It Works

1. User memulai kuis
2. Menjawab 30 pertanyaan
3. Sistem menghitung skor tiap dimensi
4. Menghasilkan kode kepribadian
5. Menampilkan hasil dalam dashboard lengkap
